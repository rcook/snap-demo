# `snap-demo`

Snap demo

## Clone repository

```
git clone https://github.com/rcook/snap-demo.git
```

## Install compiler

```
stack setup
```

## Build

```
stack build --fast
```

## Run application

```
stack exec snap-demo
```

You can view the app at [`http://localhost:8000`][view] in your browser.
